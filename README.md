# Java Innovative Projects

This repository contains three innovative Java projects developed to demonstrate practical applications of Java programming concepts such as OOP, Collections Framework, File Handling, String Processing, and Date/Time Operations.

## Projects Included

### 1. Smart Password Generator

#### Objective

A utility application that generates secure passwords using user-specific information and evaluates password strength.

#### Features

* Accepts user details such as Name, PAN, and Date of Birth
* Generates a secure password based on the provided information
* Displays the generated password
* Classifies password strength as:

  * Weak
  * Medium
  * Strong
* Supports testing with multiple users

#### Concepts Used

* Classes and Objects
* Methods
* String Manipulation
* Conditional Statements
* User Input Handling

---

### 2. Personal Expense Tracker

#### Objective

A real-life application for managing and tracking daily expenses.

#### Features

* Stores expense details including amount, category, and date
* Displays all recorded expenses
* Generates monthly expense reports
* Identifies the highest expense category
* Saves and loads data using file handling
* Supports multiple expense entries

#### Concepts Used

* Collections Framework (ArrayList, HashMap)
* File Handling
* Date and Time Operations
* Serialization
* Menu-Driven Programming

---

### 3. Text Steganography System

#### Objective

A security-based application that hides a secret message within a cover text and retrieves it later without affecting the visible content.

#### Features

* Accepts a cover text and a secret message
* Encodes and hides the message within the cover content
* Saves encoded output
* Retrieves hidden messages
* Maintains the original visible content
* Supports multiple test inputs

#### Concepts Used

* String Processing
* File Handling
* Data Hiding Techniques
* Object-Oriented Programming
